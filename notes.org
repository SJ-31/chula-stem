** Pipeline tools notes
+ Only small variant callers produce FORMAT/DP (important cause you use this to filter), so the SV callers below won't have it
  * gridss
  * delly
  * manta
+ If FACETS doesn't work for generating purity, ploidy, you could try  BubbleTree, but give it cnv data from cnvkit on threshold mode. Then use the output of bubbletree to refine the cnvkit results
  * Only problem is that BubbleTree's input is troublesome
+ [1] show that computational methods of assessing tumor purity and ploidy don't often agree with pathologists' reports, but note that this is 4 years old
